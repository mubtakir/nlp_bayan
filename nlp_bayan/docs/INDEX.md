## فهرس وثائق nlp_bayan

هذا الفهرس يجمع أهم وثائق النموذج وروابط الدخول السريعة:

- الدليل التفصيلي: NLP_BAYAN_MODEL_GUIDE_AR.md
- نظرة تصميمية: DESIGN_OVERVIEW_AR.md
- دليل الاستلام للمكمّل الذكي: HANDOFF_GUIDE_AR.md
- أمثلة التشغيل: ../examples/
- واجهة API مبسطة: ../api/simple_api.bayan

تشغيل سريع:

- مثال التوليد: `python -m bayan nlp_bayan/examples/demo_generation.bayan`
- مثال حوار بسيط: `python -m bayan nlp_bayan/examples/demo_dialogue_generation.bayan`

- كل الاختبارات: `pytest -q`
- اختبار مُحدد: `pytest -q tests/test_nlp_bayan_generation.py`
