# AI Model Links for Bayan — روابط سهلة للوصول من النماذج اللغوية

هذه الصفحة تجمع روابط "نص خام" Raw مباشرة لكي تتمكن النماذج اللغوية (ChatGPT, Claude, Gemini, DeepSeek, …) من قراءة مشروع "البيان" حتى لو كانت صفحة GitHub العادية صعبة الوصول لديهم.

## كيف تستخدم هذه الروابط؟
- لو كان النموذج يدعم التصفح: أعطه روابط raw.githubusercontent.com أدناه.
- لو لم يدعم التصفح: ارفع الملفات كمرفقات، أو أعطه ملف ZIP (انظر آخر القسم).
- بديل على GitHub العادي: أضف ?plain=1 في آخر الرابط لعرض نص بسيط.

مثال "plain":
- https://github.com/mubtakir/bayaan-lang/blob/main/docs/EXAMPLES.md?plain=1
