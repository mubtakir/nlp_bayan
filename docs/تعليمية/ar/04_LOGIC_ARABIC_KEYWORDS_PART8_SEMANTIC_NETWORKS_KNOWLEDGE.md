# المنطق في لغة البيان (٨): الشبكات الدلالية والمعرفة والمعاني

<div dir="rtl">

> في هذا الجزء ندخل إلى **عالم المعنى والمعرفة** في البيان:
> - شبكات المعاني `معنى`.
> - المعلومات المعرفية `معلومة`.
> - قواعد الاستدلال `قاعدة_استدلال`.
> - كشف التناقضات `تناقض بين`.
> - المعرفة المتطورة، الأنطولوجيا، والذاكرة الدلالية والتشابه.
>
> سنحافظ على الأمثلة عربية بالكامل في الأسماء والبيانات، مع استخدام الكلمات العربية التي يوفّرها المترجم.

## ١. شبكات المعاني: `معنى`

شبكة المعاني تمثّل **معنى كيان** (شخص، شيء، مفهوم) كقاموس علاقات.

### ١٫١ مثال بسيط: معنى طالب

```bayan
معنى محمد:
{
    "هو": "طالب",
    "يدرس": "الرياضيات",
    "عمره": 20
}

معنى علي:
{
    "هو": "طالب",
    "يدرس": "البرمجة",
    "عمره": 22
}
```

### ١٫٢ استعلام منطقي فوق شبكة المعاني

يمكننا كتابة قواعد منطقية تستخدم `معنى` كعلاقة ضمنية، مثلاً للبحث عن من يدرس مادة معيّنة:

```bayan
هجين {
    من_يدرس(?شخص, ?مادة) :- معنى ?شخص: {"يدرس": ?مادة}.
}

استعلام من_يدرس(?شخص, "الرياضيات")?
```

> الصياغة الدقيقة الداخلية لـ`معنى ?شخص: {...}` يحدّدها المترجم، لكن الفكرة التعليمية هنا أن **شبكة المعنى** يمكن استعمالها داخل قواعد منطقيّة عربية.

## ٢. معلومات معرفية: `معلومة`

`معلومة` تخزّن محتوى + سياق (زمان، مكان، مصدر، يقين).

### ٢٫١ مثال: «الأرض كروية» مع سياق عربي

```bayan
معلومة "الأرض كروية":
{
    "محتوى": {"شكل": "كروي", "قطر": 12742},
    "سياق": {
        "زمان": "2024",
        "مكان": "علم الفلك",
        "مصدر": "مرصد عربي",
        "يقين": 0.99
    }
}
```

يمكننا لاحقًا استخدام هذه المعلومة في الاستدلال، أو مقارنة يقينها مع معلومات أخرى.

## ٣. قواعد الاستدلال: `قاعدة_استدلال` و`استدل_من`

تسمح لنا **قواعد الاستدلال** باشتقاق معرفة جديدة من معلومات موجودة.

### ٣٫١ مثال: «الطالب الذي يدرس الرياضيات ذكي»

```bayan
قاعدة_استدلال "إذا كان طالب يدرس الرياضيات فهو ذكي":
{
    "if": ["يدرس", "الرياضيات"],
    "then": {"ذكي": 1}
}

معلومة "محمد يدرس الرياضيات":
{
    "محتوى": {"اسم": "محمد", "يدرس": "الرياضيات"},
    "سياق": {"يقين": 0.9}
}

استدل_من: "محمد يدرس الرياضيات"  # يستنتج أن محمد ذكي
```

الهدف هنا تربوي: أن ترى كيف تُكتَب القاعدة بالعربية، وكيف تُستخدم معلومة عربية كأساس للاستنتاج.

## ٤. كشف التناقضات: `تناقض بين`

يمكن للبيان فحص قائمة معلومات والبحث عن تناقضات بينها بناءً على درجة اليقين.

### ٤٫١ مثال: «الأرض مسطحة» ضد «الأرض كروية»

```bayan
معلومة "الأرض مسطحة":
{
    "محتوى": {"شكل": "مسطح"},
    "سياق": {"يقين": 0.1}
}

معلومة "الأرض كروية":
{
    "محتوى": {"شكل": "كروي"},
    "سياق": {"يقين": 0.99}
}

تناقض بين: ["الأرض مسطحة", "الأرض كروية"]
# يختار النظام المعلومة ذات اليقين الأعلى (الأرض كروية)
```

هذا يربط البرمجة الدلالية بما رأيناه في **الاحتمالات والتشكيك**: اليقين (0..1) يدخل في قرار أي معلومة نعتمد.

## ٥. معرفة متطورة: `معرفة_متطورة`

تسمح `معرفة_متطورة` بمتابعة كيفية تغيّر المعرفة عبر الزمن.

### ٥٫١ مثال: تطوّر فهمنا للكون

```bayan
معرفة_متطورة "فهمنا للكون":
{
    "قيمة_حالية": "نظرية الانفجار العظيم",
    "تاريخ": [
        {"زمن": "القرن 17", "قيمة": "الكون ثابت"},
        {"زمن": "القرن 20", "قيمة": "الكون يتمدد"}
    ],
    "توقع_مستقبلي": "نظريات أكثر دقة"
}
```

يمكن لاحقًا ربط هذا بعالم الأسباب (الشبكات السببية) أو بنموذج «الفكرة–الحدث–النتيجة».

## ٦. الأنطولوجيا، الذاكرة الدلالية، والتشابه

لن نتعمّق طويلًا هنا، لكن نعرض أمثلة عربية سريعة توضّح الصورة.

### ٦٫١ أنطولوجيا بسيطة للكائنات الحية

```bayan
أنطولوجيا "الكائنات الحية":
{
    "جذر": "كائن حي",
    "تصنيف": {
        "حيوان": {
            "ثدييات": ["قط", "كلب", "إنسان"],
            "طيور": ["عصفور", "نسر"]
        },
        "نبات": {
            "أشجار": ["نخلة", "صنوبر"],
            "أزهار": ["وردة", "ياسمين"]
        }
    }
}
```

### ٦٫٢ ذاكرة دلالية شخصية

```bayan
ذاكرة_دلالية "ذكرياتي":
{
    "تخزين": {
        "حدث1": "تخرجت من الجامعة",
        "حدث2": "زرت مدينة مكة",
        "حدث3": "تعلمت لغة البيان"
    },
    "استرجاع": "حدث1"
}
```

### ٦٫٣ تشابه دلالي بين مفهومين

```bayan
مفهوم "قط":
{
    "نوع": "حيوان",
    "حجم": "صغير",
    "صوت": "مواء"
}

مفهوم "كلب":
{
    "نوع": "حيوان",
    "حجم": "متوسط",
    "صوت": "نباح"
}

درجة_تشابه = تشابه("قط", "كلب")  # قيمة تقريبية بين 0 و 1
```

- هنا نرى **تشابهًا دلاليًا** مبنيًا على السمات (النوع، الحجم، الصوت...).
- في الجزء (٩) سنرى طبقة تشابه عامة `similar/5` يمكن أن تستفيد منها دالة `تشابه` هذه،
  لربط المترادفات والمعاني في شبكات `معنى` و`مفهوم`.

## ٧. ربط البرمجة الدلالية ببقية الأجزاء

- **معنى / معلومة / معرفة_متطورة** تقدّم طبقة «المعنى والمعرفة» فوق:
  - المنطق الكلاسيكي (الحقائق والقواعد).
  - الاحتمالات والتشكيك (درجات اليقين).
  - الكيانات والحالات والأفعال (ما يحدث في العالم).
  - الشبكات السببية (لماذا يحدث).
- كل ذلك بواجهة **عربية بالكامل** تقريبًا، مع إبقاء التفاصيل التقنية داخلية.

بهذا الجزء، اكتملت الصورة الكبرى: ليس لدينا فقط كائنات وحالات وأسباب واحتمالات، بل أيضًا **شبكات معاني ومعرفة دلالية** يمكن الاستدلال منها، كشف تناقضاتها، وتتبع تطورها عبر الزمن.
