# الجزء الثالث (الجزء 2): قواعد مركبة وعودية

<div dir="rtl">

> (قريبًا) سيتم نقل قسم القواعد المركبة والعودية والأمثلة المرتبطة.



## 6. القواعد المركبة

```bayan
هجين {
    أب("أحمد", "محمد"). أب("محمد", "علي").
    أب("فاطمة", "سارة"). أب("سارة", "ليان").

    سلف(?سلف, ?خلف) :- أب(?سلف, ?خلف).
    سلف(?سلف, ?خلف_بعيد) :- أب(?سلف, ?خلف_متوسط), سلف(?خلف_متوسط, ?خلف_بعيد).
}
```

## 7. العودية (Recursion)

```bayan
هجين {
    حافة(a, b). حافة(b, c). حافة(c, d).

    مسار(?من, ?إلى) :- حافة(?من, ?إلى).
    مسار(?من, ?إلى) :- حافة(?من, ?وسيط), مسار(?وسيط, ?إلى).

    اطبع(استعلام مسار(a, d)?)
}
```

## 8. القوائم في المنطق

```bayan
هجين {
    عنصر_في_قائمة(?عنصر, [ ?عنصر | ?_ ]).
    عنصر_في_قائمة(?عنصر, [ ?_ | ?باقي ]) :- عنصر_في_قائمة(?عنصر, ?باقي).
}
```

## 9. العمليات المنطقية (AND/OR/NOT)

```bayan
هجين {
    ذكر("أحمد"). أنثى("فاطمة").
    يحب("أحمد", "شاي"). يحب("فاطمة", "قهوة").

    محب_الشاي(?شخص) :- يحب(?شخص, "شاي").
    محب_القهوة(?شخص) :- يحب(?شخص, "قهوة").

    # AND
    يحب_كليهما(?شخص) :- محب_الشاي(?شخص), محب_القهوة(?شخص).

    # OR
    يحب_أحدهما(?شخص) :- محب_الشاي(?شخص).
    يحب_أحدهما(?شخص) :- محب_القهوة(?شخص).
}
```
