# الذكاء العصبي الرمزي في لغة بيان
# Neuro-Symbolic AI in Bayan

لغة بيان تجمع بين قوتين عظيمتين: **المنطق الصارم** (Symbolic Logic) و **الحدس العصبي** (Neural Intuition). هذا الدمج يسمح للنظام بفهم المعاني وليس فقط مطابقة الكلمات.

Bayan combines two great powers: **Symbolic Logic** and **Neural Intuition**. This fusion allows the system to understand semantics, not just match keywords.

---

## 1. المحرك العصبي (Neural Engine)

المحرك العصبي هو الجزء المسؤول عن فهم "المعنى" في النصوص. يستخدم نماذج التعلم العميق (Deep Learning) لتحويل الكلمات والجمل إلى متجهات رياضية (Vectors).

The Neural Engine is responsible for understanding "meaning" in text. It uses Deep Learning models to transform words and sentences into mathematical vectors.

### كيفية استخدامه (Usage):

```python
from bayan.bayan.neural.neural_engine import NeuralEngine

# تهيئة المحرك
# Initialize the engine
neural = NeuralEngine()

# الحصول على التضمين (Embedding)
# Get embedding
vector = neural.embed("الشمس ساطعة")
```

---

## 2. جسر التنسور (Tensor Bridge)

جسر التنسور هو حلقة الوصل بين عالم المنطق (Facts/Concepts) وعالم الأعصاب. يقوم بتحويل الحقائق المنطقية إلى نصوص يمكن للمحرك العصبي فهمها.

The Tensor Bridge connects the Symbolic world (Facts/Concepts) with the Neural world. It converts logical facts into text that the Neural Engine can understand.

### الوظائف (Functions):

*   `embed_fact(fact)`: تحويل حقيقة منطقية إلى متجه.
*   `find_similar_facts(query, facts)`: البحث عن حقائق مشابهة في المعنى.

---

## 3. البحث الدلالي (Semantic Search)

أهم ميزة في هذا النظام هي القدرة على البحث بالمعنى. بدلاً من البحث عن كلمة مطابقة تماماً، يمكنك البحث عن المفهوم.

The most important feature is Semantic Search. Instead of searching for an exact word match, you can search for the concept.

### مثال (Example):

تخيل أن لديك في قاعدة المعرفة:
`Fact: is_hot(Sun)` (الشمس حارة)

إذا بحثت منطقياً عن "burning" (احتراق)، لن تجد شيئاً لأن الكلمة مختلفة.
ولكن مع البحث العصبي:

```python
# البحث عن "شيء يحترق"
# Search for "something burning"
results = engine.neural_search("Something burning")

# النتيجة:
# Result: Sun is is_hot (Score: 0.85)
```

فهم النظام أن "الحرارة" و "الاحتراق" مرتبطان دلالياً!
The system understood that "heat" and "burning" are semantically related!

With this architecture, Bayan is no longer just a programming language or a traditional expert system, but a **Hybrid Mind** combining the precision of logic with the flexibility of human understanding.
