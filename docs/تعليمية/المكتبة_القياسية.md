# مكتبة بيان القياسية
# Bayan Standard Library

تم توسيع مكتبة بيان القياسية لتشمل أدوات أساسية لكل مبرمج.
Bayan Standard Library has been expanded to include essential tools for every programmer.

---

## 1. مكتبة العشوائية (Random Library)
`bayan.bayan.stdlib.random_lib`

التعامل مع الأرقام العشوائية والاحتمالات.

| الوظيفة (Function) | الاسم العربي | الوصف (Description) |
| :--- | :--- | :--- |
| `randint(a, b)` | `رقم_صحيح` | رقم عشوائي بين a و b. |
| `choice(seq)` | `اختر` | اختيار عنصر عشوائي من قائمة. |
| `shuffle(list)` | `خلط` | إعادة ترتيب القائمة عشوائياً. |
| `random_float()` | `عشوائي` | رقم عشري بين 0.0 و 1.0. |

---

## 2. مكتبة النظام (System Library)
`bayan.bayan.stdlib.system_lib`

التحكم في النظام وتشغيل الأوامر.

| الوظيفة (Function) | الاسم العربي | الوصف (Description) |
| :--- | :--- | :--- |
| `execute(cmd)` | `نفذ` | تشغيل أمر طرفية (Shell Command). |
| `get_env(key)` | `متغير_بيئة` | قراءة متغيرات البيئة. |
| `exit(code)` | `خروج` | إنهاء البرنامج. |
| `get_platform()` | `نظام_تشغيل` | معرفة نوع النظام (Windows/Linux). |

---

## 3. مكتبة CSV (CSV Library)
`bayan.bayan.stdlib.csv_lib`

قراءة وكتابة البيانات المجدولة.

| الوظيفة (Function) | الاسم العربي | الوصف (Description) |
| :--- | :--- | :--- |
| `read_csv(path)` | `اقرأ_csv` | قراءة ملف كقائمة قواميس. |
| `write_csv(path, data)` | `اكتب_csv` | كتابة قائمة قواميس لملف. |

---

## 4. مكتبات أخرى (Other Libraries)

*   **IO Library:** `اقرأ_ملف`, `اكتب_ملف`.
*   **Math Library:** `جذر`, `جيب`, `باي`.
*   **HTTP Library:** `احصل` (GET), `أرسل` (POST).
*   **DateTime Library:** `الآن`, `تاريخ`, `وقت`.

المكتبة مصممة لتكون "ثنائية اللغة" (Bilingual)، يمكنك استخدام الأسماء الانجليزية أو العربية حسب تفضيلك.
The library is designed to be Bilingual, you can use English or Arabic names as you prefer.
