# أدوات المطورين في بيان
# Bayan Developer Tools

لجعل عملية التطوير أسهل وأكثر متعة، قمنا بإضافة مجموعة من الأدوات التفاعلية والمرئية.

To make development easier and more fun, we added a set of interactive and visual tools.

---

## 1. القشرة التفاعلية (Interactive Console)
`bayan/bayan/console.py`

واجهة سطر أوامر (REPL) تتيح لك التفاعل مع النظام المنطقي مباشرة.

### كيفية التشغيل (How to run):
```bash
python3 -m bayan.bayan.console
```

### الأوامر (Commands):
*   `fact loves(Ali, Reading)`: إضافة حقيقة.
*   `search Ali`: بحث دلالي عصبي.
*   `world Dream`: الانتقال لعالم آخر.

---

## 2. لوحة القيادة المرئية (Web Dashboard)
`bayan/bayan/web_dashboard.py`

خادم ويب محلي يعرض حالة النظام الداخلي ورسوم بيانية للمعرفة.

### كيفية التشغيل (How to run):
```bash
python3 -m bayan.bayan.web_dashboard
```
ثم افتح `http://localhost:8080` في المتصفح.

### الميزات (Features):
*   عرض عدد الحقائق والعالم النشط.
*   رسم بياني (Knowledge Graph) للعلاقات بين المفاهيم.
*   مراقبة الحالة العصبية للنظام.

---

## 3. تطبيق الويب المصغر (Micro-Web App)
`bayan/examples/app_demo.py`

مثال عملي يوضح كيفية استخدام مكتبات بيان لبناء واجهة برمجية (API).

### كيفية التشغيل (How to run):
```bash
python3 bayan/examples/app_demo.py
```
ثم جرب الرابط: `http://localhost:9000/search?q=text`

يوضح هذا المثال كيف تعمل مكتبة `http_lib` مع `IstinbatEngine` لخدمة الطلبات.
