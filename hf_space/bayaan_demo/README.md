# Bayaan â€” Interactive Demo (Gradio Space)

This Space provides a lightweight, bilingual (AR/EN) demo for the Bayaan project:
- Browse the Bayaan alignment dataset on Hugging Face
- See precomputed evaluation metrics (v1.2)
- Copy/paste Quickstart code snippets to use the dataset locally
- Validate Bayan code syntax using the full HybridParser (vendored, with fallback)


Notes:
- This demo does not execute Bayaan code; it focuses on dataset exploration and showing evaluation results.
- For full evaluation locally, see the project repository and eval framework docs.
