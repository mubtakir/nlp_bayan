# تقرير: مشكلة الصيغة في ملفات الأمثلة
# Syntax Inconsistency Report

## المشكلة / Problem

**وجدت أن العديد من ملفات الأمثلة تستخدم صيغة Python بدلاً من صيغة Bayan الصحيحة.**

### الصيغة الخاطئة (Python)
```python
if condition:
    statement
    
while condition:
    statement
    
for item in list:
    statement
    
def function():
    statement
```

### الصيغة الصحيحة (Bayan)
```bayan
if (condition) {
    statement
}

while (condition) {
    statement
}

for (item in list) {
    statement
}

fn function() {
    statement
}
```

## التأثير / Impact

### ⚠️ خطورة عالية
1. **التباس للمستخدمين:** المستخدمون الجدد سيتعلمون صيغة خاطئة
2. **عدم الاتساق:** بعض الملفات تستخدم `{}` وبعضها يستخدم `:`
3. **أخطاء محتملة:** الكود قد لا يعمل إذا كان المفسر يتوقع `{}`
