# Change Log

All notable changes to the "bayan" extension will be documented in this file.

## [0.1.0] - 2025-11-25

### Added
- Initial release of Bayan Language Support extension
- Comprehensive syntax highlighting for 200+ keywords (English and Arabic)
- Support for logical variables (`?X`, `?variable`)
- Syntax highlighting categories:
  - Control flow keywords (`if`, `for`, `while`, etc.)
  - Logic programming keywords (`hybrid`, `fact`, `rule`, `query`)
  - Entity system keywords (`entity`, `apply`, `concept`)
  - Temporal keywords (`temporal`, `schedule`, `every`)
  - Semantic programming keywords (`meaning`, `knowledge`, `similarity`)
  - Cognitive semantics keywords
  - Causal reasoning keywords
  - Existential model keywords
  - Spatial and life domain keywords
- 20+ code snippets for common patterns:
  - Function and class definitions
  - Control flow structures
  - Hybrid logic blocks
  - Entity definitions
  - Fact and rule declarations
  - Medical expert system template
  - Arabic snippets
- Language configuration:
  - Comment support (`#`)
  - Bracket matching
  - Auto-closing pairs
  - Code folding
- File associations for `.bayan` and `.by` extensions
- Bilingual support (Arabic and English)
- Triple-quoted string support
- Operator highlighting (pipeline `|>`, composition `>>`, etc.)

### Features
- Full RTL (Right-to-Left) support for Arabic code
- Distinct syntax colors for different keyword categories
- Function and class name highlighting
- Number and string literal highlighting
- Escape sequence support in strings
- Comment highlighting
